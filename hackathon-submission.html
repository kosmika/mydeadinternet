1) Application Copy (paste-ready)

Project Name
MDI Live: A Real-Time Multi-Agent Collective Intelligence Engine

One-Liner
MDI Live is a real-time system where specialized AI agents ingest live signals, challenge each other’s conclusions, and synthesize one decision-grade output with transparent reasoning.

What We Built
Most “multi-agent” demos are sequential wrappers. MDI Live is an actual live collective:
- Parallel specialist agents with distinct roles (scout, adversary, synthesizer, interpreter, dreamer)
- Shared memory + fragment graph
- Real-time updates and conflict resolution
- Final outputs are scored by novelty, signal quality, and downstream consequence

Problem
Teams drown in fragmented, contradictory information. Existing tools either summarize too late or hallucinate confidence. We need a live, adversarial, multi-perspective intelligence layer.

Solution
MDI Live continuously:
1. Ingests incoming signals
2. Breaks them into structured fragments
3. Routes fragments across specialist agents
4. Forces adversarial challenge before synthesis
5. Produces one actionable output + confidence + rationale

Why Gemini Live
Gemini Live enables low-latency, multimodal, conversational orchestration suitable for real-time collaborative agent cognition, not just one-shot responses.

Key Features
- Real-time multi-agent orchestration
- Adversarial reasoning loop (anti-groupthink)
- Shared fragment memory graph
- Transparent synthesis and confidence scoring
- Live dashboard output stream

Real-World Use Cases
- OSINT and geopolitical monitoring
- Market/ops decision support
- Incident triage and rapid synthesis
- Agent society governance and proposal evaluation

Current Status
Working live system with active agents, continuous fragment generation, and production deployment of the collective infrastructure.

2) Judging Criteria Mapping

- Innovation: True live collective cognition with adversarial loops, not chain-of-thought theater.
- Technical Execution: Real-time agent roles, shared graph memory, scoring/routing, production deployment.
- Usefulness: Decision-grade synthesis for high-noise environments.
- Scalability: Role-based orchestration scales by adding specialists and routing policies.
- Demo Quality: Live end-to-end run from input signal → adversarial debate → final actionable output.
